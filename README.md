Wanderlust
Wanderlust is a full-stack web application designed to help users explore, share, and book unique travel accommodations. 
The platform supports property listings, booking, user authentication, and location features, aiming to provide a seamless experience for both travelers and property hosts.

Features
User authentication for secure sign up and login
Browse, add, update, and delete property listings
Booking system for accommodations
Responsive UI for mobile and desktop
Interactive and intuitive user interface with EJS templating
Modern design using CSS3
Data management with MongoDB (if implemented)
MVC architecture for code clarity

Tech Stack
Frontend: HTML5, CSS3, JavaScript, EJS
Backend: Node.js, Express.js
Database: MongoDB
Other Tools: Passport.js (authentication), Cloudinary (image upload), Mapbox, MongoDB Atlas(Cloud Configuration).
